# Classroom submission checklist

Use this checklist before clicking "Hand in" on Google Classroom.

## 1. PPTX (mandatory format: .pptx)

- [ ] At least **50 slides** — current deck has **55**.
- [ ] All text in **English**.
- [ ] At least **5 AI-generated images** inserted (see `docs/ai_prompts.md`).
- [ ] GitHub repository link visible on slide 4 and slide 55.
- [ ] List of all AI-powered tools used is on slide 5 (and slide 54 recap).
- [ ] Tutorial section: ✔ background, JEPA family, I-JEPA, code walkthrough.
- [ ] Detailed illustrative example: ✔ CIFAR-10 pretraining + linear probing.
- [ ] File saved as `JEPA_Presentation.pptx` (NOT .pdf, .key, .gslides).

## 2. Video (mandatory: 20 minutes EXACTLY)

- [ ] Recorded at 1080p / 30 fps.
- [ ] Length between **19:55 and 20:05** (re-record if drift > 10 s).
- [ ] Detailed live demo of the code (3 segments: see `docs/video_script.md`).
- [ ] At least **2 AI-generated video clips** embedded.
  - Clip #1 inserted at ≈ 4:00 (20 s)
  - Clip #2 inserted at ≈ 17:30 (20 s)
- [ ] Uploaded to **Google Drive**, share visibility = **Anyone with the link**.
- [ ] Drive link added to the **PPTX final slide** before submitting.

## 3. GitHub repository (mandatory: public)

- [ ] Repository is **public** (not "private — invite collaborators").
- [ ] README rendered correctly on the repo home page.
- [ ] `requirements.txt` present and pinned.
- [ ] `LICENSE` present (MIT).
- [ ] No `data/` or `checkpoints/` committed (covered by `.gitignore`).
- [ ] Final commit pushed before the deadline.

## 4. Items to upload on Classroom

| Item                          | Format            |
|-------------------------------|-------------------|
| Presentation                  | `.pptx`           |
| Video link (Google Drive)     | URL (text field)  |
| GitHub repository link        | URL (text field)  |

## 5. Quick "looks good" sanity test

- [ ] Open the PPTX in PowerPoint (not just LibreOffice) — make sure layouts
      survived.
- [ ] Click the GitHub link in the final slide — it must open the **public**
      page in incognito mode.
- [ ] Click the Drive link in the final slide — incognito mode must be able
      to play the video without requesting access.

If all boxes are ticked, you are ready to hand in.
